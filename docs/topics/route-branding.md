# 노선 배지 색상

> 이 문서는 **현재 상태**를 서술한다. 변경 이력은 [devlog.md](../devlog.md)를 참조한다.
> 마지막 갱신: 2026-07-12

## 현행 규칙

- 노선 배지와 상세 화면의 노선도 버튼은 `lib/utils/route_branding.dart`의 공용
  분류를 사용한다.
- 분류는 노선명과 설명 문자열에서 `직행`, `급행`, `순환`, `간선`, `지선`, `출근맞춤`,
  `군위`, `투어`, `DRT`를 우선 탐지한다.
- `직행`은 흰 배경에 빨간 글자와 빨간 테두리를 사용한다.
- `급행`, `순환`, `간선`, `지선`, `출근맞춤`, `군위`, `투어`, `DRT`는 지정된 단색 배경에
  흰 글자를 사용한다.
- 분류 문자열이 없으면 기존 노선 유형/테마 색으로 폴백한다.
